'''
任务三：练习使用 Pyinstaller 打包为可执行程序
把任务二中写好的 SSH 命令行脚本，使用 pyinstaller 打包成脱离 Python 环境也能运行的独立可执行程序。

注意：此任务请在 Windows 环境下执行！因为 Pyinstaller 打包的程序是与操作系统绑定的，在 Windows 下打包才能生成 .exe 文件。

重要提示：由于 paramiko 依赖了许多底层加密库，Pyinstaller 有时无法自动发现这些依赖，导致打包后的程序运行报错。因此在打包时需要使用 --hidden-import 手动指定隐藏依赖。

完整的打包命令（在 PowerShell 中执行）：

pyinstaller --onefile `
  --hidden-import=paramiko `
  --hidden-import=cryptography `
  --hidden-import=cryptography.hazmat.primitives.asymmetric.ed25519 `
  --hidden-import=cryptography.hazmat.backends.openssl `
  --hidden-import=bcrypt `
  day14_task02_ssh_argparse.py
打包完成后，进入 dist/ 目录，在 Windows 的 PowerShell 中测试运行你生成的 exe 文件：

PS C:\> .\dist\day14_task02_ssh_argparse.exe -i 196.21.5.211 -u admin -p Cisc0123 -c "show ip inter brief"

Interface              IP-Address      OK? Method Status                Protocol
GigabitEthernet1       196.21.5.211    YES NVRAM  up                    up
GigabitEthernet2       172.16.1.12     YES manual administratively down down
GigabitEthernet3       unassigned      YES TFTP   administratively down down
Loopback13             13.13.13.13     YES manual up                    up
'''
